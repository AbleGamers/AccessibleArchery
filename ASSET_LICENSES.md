# Asset Licenses

Code in this repository is licensed under [MIT](LICENSE). Code licenses fit
creative assets poorly, so **art, audio, models, and fonts are tracked
separately here.**

## Default for contributed assets

- Preferred: **CC0-1.0** (public domain dedication) or **CC-BY-4.0** (attribution).
- Anything contributed must be **legally redistributable in an open-source repo**.

## Hard rules

- ❌ No Unity Asset Store / proprietary marketplace assets.
- ❌ No "free for personal use only" art, fonts, or audio.
- ⚠️ **AI-generated assets:** verify the generator's terms permit redistribution
  under our license *and* that outputs are free of third-party claims. Record the
  tool and license below.
- ✅ When in doubt, leave it out — use a placeholder and open an issue.
